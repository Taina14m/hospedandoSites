Passos para Hospedar um Projeto no GitHub pages

  1. Crie um repositório no GitHub com o mesmo nome de seu projeto
  2. No terminal, inicialize o repositório local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit com git commit -m "first commit"
  5. Conecte ao repositório remoto com:
    git remote add origin https://github.com/SEU_USUARIO_/NOME_DO_REPOSITORIO.git
  6. Envie arquivos com git push -u
